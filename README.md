# Atherion - Wisdom Quiz

A beautiful ancient Greek-themed wisdom quiz game built with React (no build step required).

## 🏛️ Features

- 10 thought-provoking wisdom questions
- Ancient Greek aesthetic with parchment-style UI
- Interactive quiz with immediate feedback
- Progress tracking with visual indicators
- Prize code system for educational use
- Fully responsive design
- No build tools required - runs directly in the browser

## 🚀 Live Demo

[Play the Quiz](https://your-vercel-url.vercel.app)

## 🛠️ Local Development

1. Clone the repository:
```bash
git clone https://github.com/your-username/atherion-wisdom-quiz.git
cd atherion-wisdom-quiz
```

2. Start a local server:
```bash
# Using Python
python -m http.server 3000

# Or using Node.js
npx serve .

# Or using any other static file server
```

3. Open your browser and navigate to `http://localhost:3000`

## 🎨 Technology Stack

- **React 18** - UI library (loaded via CDN)
- **Babel Standalone** - JSX transformation
- **CSS3** - Styling with gradients and animations
- **Vanilla JavaScript** - No build tools required

## 🎯 Educational Use

This quiz is designed for educational purposes. Students receive a prize code upon completion that can be presented to teachers for rewards or recognition.
